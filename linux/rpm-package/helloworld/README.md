# helloworld example
In this directory, run this:
```
copy hello-world.sh ~/
cp -r rpmbulid ~/
```

In $HOME directory, run this:
```
rpmbuild -bb rpmbuild/SPECS/hellowworld.spec
```

Another way is to create a link to rpmbuild in home directory.
